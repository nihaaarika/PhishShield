# PhishShield
ScamGuard AI is a multi-class NLP-based scam message detection system that classifies messages into phishing, OTP fraud, lottery scams, job scams, or safe content. The system provides a risk score, explainable threat analysis, and safety recommendations to improve cybersecurity awareness among students.

An end-to-end NLP pipeline using TF-IDF and Logistic Regression for multi-class scam detection with explainable AI, real-time risk scoring, and an interactive Streamlit interface.
